Platform Overview

Baitlab is a unified AI operating layer for businesses and individuals. The platform combines voice AI, workflow automation, and intelligent integrations into a single, easy-to-use interface. No coding required for core use cases — advanced customization available via API.
REST API

All Baitlab functionality is accessible via a clean REST API. Authenticate with Bearer tokens, receive webhook events for call outcomes, and trigger workflows programmatically. API rate limits: 1,000 req/min on Pro, unlimited on Enterprise.

POST https://api.baitlab.io/v1/calls
JavaScript SDK

Install via npm: npm install @baitlab/sdk

The SDK provides typed wrappers around all API endpoints, real-time event streaming via WebSocket, and React/Vue component bindings for the live chat widget.
